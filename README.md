# NotificationService
NotificationService + Authentication Service

- STS(JWT base) + Identity User
- Api Gatway with Ocelot
- Send Email and SMS
- Microservice
- RabbitMq
- DDD and CQRS and Mediator
- ELK stack for logging
- pub/sub pattern
- dotnetcore
- Minimal Api
